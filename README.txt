To run the Core Interpreter Project you must run the following command in the command prompt: "python main.py *file1* *file1Data*"

*file1* is the file which contains for Core program you want to run.

*file1Data* is the file which contains the integers (positive or negative) that will be used for "read".
NOTE: No line in the *file1data* file cannot be only a newline (i.e "\n") such as the following:

Example 1:
------
1

2
3
------

Example 2:
-------

1
2
-------

Example 3:
-------
1


-------

These are three example of what the datafile cannot contain.
